Best Results

A few constraints make the output much more believable.

  • Target short sounds, not full arrangements.
  • Generate presets inspired by the source, not exact reconstruction.
  • Use mode hints because the same spectrum can imply different synth behavior.
  • Vital is supported first, with Serum and Pigments planned next.
