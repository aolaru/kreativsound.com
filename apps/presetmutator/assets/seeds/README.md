# Seed Presets

Place bundled source presets here, grouped by synth.

## Vital

- `assets/seeds/vital/raw/`
  Original `.vital` files, unchanged
- `assets/seeds/vital/metadata/`
  Per-preset metadata files to add later
- `assets/seeds/vital/review/`
  Notes, parsed samples, and inspection outputs

## Rules

- do not modify the original raw preset files
- keep original filenames unless there is a real conflict
- metadata should be stored separately from the raw preset file
