# Kreativ Sound Logo System

Kreativ Sound uses the red K as the compact mark and the full wordmark when brand clarity matters. Keep the system simple and consistent.

## Assets

- `logo-128.svg`: primary compact K mark for tight spaces, thumbnails, and social fallback images.
- `kreativ-sound-wordmark.svg`: primary site and marketplace lockup when there is enough horizontal room.
- `kreativ-sound-product-badge.svg`: product-cover and marketplace badge for presets, samples, and tools.
- `favicon.svg`, `favicon-16x16.png`, `favicon-32x32.png`, `apple-touch-icon.png`: browser and device icons.

## Usage Rules

- Use the wordmark for headers, press kits, marketplace profiles, and product descriptions where the name needs to be read.
- Use the K mark only when space is tight or the Kreativ Sound name is already visible nearby.
- Use the product badge on covers, screenshots, and Gumroad/Lemon Squeezy/Etsy product media when a small maker mark is needed.
- Do not stretch, rotate, recolor, or place the logo on low-contrast backgrounds.
- Keep clear space around the mark equal to at least one quarter of the K square.

## Core Colors

- Dark base: `#0d1018`
- Deep red: `#990000`
- Active red: `#c91818`
- Soft red highlight: `#e05b5b`
- Light text: `#f7ecec`
